Build a complete React Native (Expo) mobile application named “NIDHI” targeted at Tamil Nadu families with low to moderate financial literacy, including rural and first-time smartphone users. The app must be fully bilingual (Tamil default + English) using react-i18next with JSON translations and allow language switching anytime. Use Stack Navigation for onboarding/details and Bottom Tab Navigation for the main app. Onboarding flow must include: Splash screen → Language selection → Welcome/Purpose → Optional Family Setup (skip allowed). Core tabs must be: Home (family finance dashboard with quick cards), Calculators (SIP, EMI, Compound Interest, Savings Goal, ROI, Rule of 72, TVM; each with “What is this”, “Why useful”, simple inputs, visual graph output, Tamil explanation toggle), Schemes (Tamil Nadu government schemes with Tamil-first names, eligibility, benefits, documents, how to apply, estimated savings, and eligibility checker based on age/income/gender/occupation), Learn (beginner finance lessons in simple spoken Tamil and English with examples and calculator links), and More (Family Members, Banks in Tamil Nadu, Gold & Silver Prices, Settings, Disclaimer). Family member management must allow adding 1–N members with name, age, relationship, optional income/occupation and enable family-level calculations and scheme eligibility checks. Include high-priority schemes such as CM Comprehensive Health Insurance Scheme, First Graduate Scheme, Post-Matric Scholarship, Moovalur Ramamirtham Ammaiyar Scheme, Mahalir Urimai Thogai, Dr. Muthulakshmi Reddy Maternity Scheme, Marriage Assistance Schemes, Farmer Crop Insurance, Farmer Pension, Old Age Pension, Sukanya Samriddhi Yojana, Senior Citizen Savings Scheme, and TN Skill Development schemes. Add a Banks section showing indicative savings/FD/loan interest rates, minimum balance, card benefits, and 2–3 bank comparison with a clear disclaimer. Integrate a Gold & Silver price tracker with Tamil Nadu pricing, daily API updates, trend graph, and buy/sell tip. UX must be extremely beginner-friendly with big buttons, minimal text, icons, high contrast mode, smooth animations (Reanimated/Lottie), offline access for schemes, and no mandatory login; all data stored locally using AsyncStorage, with future-ready hooks for Firebase/backend and APIs. Include smart reminders (SIP, savings goals, scheme deadlines), strict privacy messaging, and a disclaimer stating educational purpose only. The app must be clean, scalable, Tamil-first, socially impactful, and production-ready.
